Description :
Navigation Bar Features:
Home: Quickly return to the main dashboard or home screen.
Browse All Apps: A dedicated navigation button to view the entire collection of apps.
Installed Apps Manager: A specific section that identifies and displays only the apps currently installed on your device.

App Features:
Contribute on GitHub: A dedicated button that redirects you to my GitHub profile to explore projects or contribute to the code.
Google Play Integration: Easily access the app's official page on the Google Play Store with a single click.
App Store Access: Provides a direct link to the App Store for quick downloads and updates.
Show All Apps: A "Show All" button that displays a comprehensive list of all available applications in one place

Key Features & Functionality:
Featured Apps on Home: Highlights the 8 most popular apps on the home screen for quick access.
Detailed App View: Click on any app card to view comprehensive app details and specifications.
Live Search with Instant Feedback: Features a real-time search bar that filters apps as you type. If no results are found, it displays a "No Apps Found" message.
Smart Install Logic: Once an app is installed, the Install button is automatically disabled to prevent duplicate actions.
Seamless Uninstallation: Easily manage your device by uninstalling apps directly through the interface.
Persistent Local Storage: Installed apps are saved and managed using Local Storage, ensuring your data is retained even after closing the app.
Installed Apps Tracker: A dedicated section to view, fetch, and remove all currently installed applications from your list.
Dynamic Search & Error Handling: Includes a Live Search Bar that updates results instantly. If no app matches, it displays "App Not Found," and for invalid navigation, it shows a "Page Not Found" screen.
Real-time Notifications: Displays instant Toast Messages for every successful installation and uninstallation to keep the user informed.
